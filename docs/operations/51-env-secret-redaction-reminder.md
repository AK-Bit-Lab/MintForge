# Env secret redaction reminder

Redact secrets from logs, screenshots, and incident threads by default.
Fast redaction habits prevent accidental credential exposure.

Scrub secrets from screenshots, terminal history, and exported HAR files before sharing troubleshooting artifacts.

Check logs and screenshots for accidental secret exposure before sharing incident threads.

Scan shared screenshots for API keys, wallet balances, and browser profile names before posting in public channels.

Escalate immediately when an exposed secret is discovered in shared logs, even if access appears limited.

Verification evidence: document where redacted artifacts are stored and who approved the redaction.
